# 评测与测试

本文档用于统一项目测试与 AI 能力评测实践。

## 测试命令

- 本地测试：`just test`
- 全量测试：`just test all`
- 真实依赖测试：`just test real`

## 测试分层建议

- **单元测试**：聚焦函数级逻辑，快速反馈。
- **集成测试**：覆盖模块协作边界。
- **真实环境测试**：针对依赖外部 API 的能力进行验证。

## AI 评测建议

- 建立黄金样本集并定期回归。
- 记录模型版本、Prompt 版本、评测时间和指标。
- 在 CI 中设置最低质量阈值。

## 建议指标

- 正确率或任务完成率
- 响应时延
- 稳定性与失败率
- 成本指标（按调用量或 token 统计）
