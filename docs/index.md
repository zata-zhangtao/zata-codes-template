# AI Agent 项目文档

本项目是一个面向 AI Agent 与通用 Python 工程实践的模板仓库，内置了配置管理、日志能力、数据库连接示例、爬虫基础组件与模型加载工具，目标是帮助你快速搭建可维护、可扩展的智能应用工程。

## 项目目标

- 提供一个开箱即用的 Python 模板，减少重复脚手架工作。
- 统一配置、日志、测试、Hook 与文档规范，降低协作成本。
- 为 AI Agent 相关能力预留清晰扩展点，如模型管理、Prompt 管理与评测流程。

## 核心特性

- 基于 `uv` + `just` 的开发工作流。
- `pydantic-settings` 驱动的分层配置体系（环境变量 + TOML + 默认值）。
- 可复用的 `utils` 组件，包括日志、数据库和辅助函数。
- `crawler` 子模块提供爬虫基类与数据库模型示例。
- `ai_agent` 子模块提供模型配置加载与实例化能力。
- 基于 MkDocs Material 的工程化文档站点。

## 快速安装

```bash
just dev
```
